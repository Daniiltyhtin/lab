Самостоятельное задание

Если в контейнере не хватает места для расположения всех элементов без изменения размеров, то свойство flex-shrink указывает, в каких пропорциях элемент будет уменьшаться.
flex-grow и flex-shrink — это CSS-свойства, определяющие, какое количество свободного пространства флекс-контейнера может занимать текущий элемент.
Свойство position определяет тип метода позиционирования, используемого для элемента (статический, относительный, фиксированный, абсолютный или липкий).
Свойство float используется для позиционирования и форматирования содержимого, например, позволяет изображению перемещаться слева от текста в контейнере.
Свойство CSS box-sizing позволяет нам включать отступы и границы в общую ширину и высоту элемента.
Атрибут HTML class используется для указания класса для HTML-элемента.
Многостолбцовый макет CSS позволяет легко определять несколько столбцов текста - точно так же, как в газетах.
CSS позволяет добавлять несколько фоновых изображений для элемента с помощью свойства background-image.
Свойство font-size задает размер текста.
Свойства границы CSS позволяют указать стиль, ширину и цвет границы элемента.
